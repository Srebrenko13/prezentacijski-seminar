Evo rješenja zadataka iz datoteke 2026 derivacije.png:Zadatak (a)Po definiciji derivacije izvedite derivaciju funkcije $f(x) = \ln^2 x$.Definicija derivacije u točki $x$ glasi:$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$Uvrštavanje funkcije:$$f'(x) = \lim_{h \to 0} \frac{\ln^2(x+h) - \ln^2 x}{h}$$Rastavljanje razlike kvadrata ($a^2 - b^2 = (a-b)(a+b)$):$$f'(x) = \lim_{h \to 0} \frac{(\ln(x+h) - \ln x) \cdot (\ln(x+h) + \ln x)}{h}$$Primjena svojstva logaritma ($\ln a - \ln b = \ln \frac{a}{b}$):$$f'(x) = \lim_{h \to 0} \left[ \frac{\ln\left(\frac{x+h}{x}\right)}{h} \cdot (\ln(x+h) + \ln x) \right]$$$$f'(x) = \lim_{h \to 0} \left[ \frac{\ln\left(1 + \frac{h}{x}\right)}{h} \right] \cdot \lim_{h \to 0} (\ln(x+h) + \ln x)$$Sređivanje prvog limesa (pomoću poznatog limesa $\lim_{t \to 0} \frac{\ln(1+t)}{t} = 1$):U prvom dijelu uvedemo zamjenu tako da dobijemo $\frac{h}{x}$ u nazivniku:$$\lim_{h \to 0} \frac{\ln(1 + \frac{h}{x})}{\frac{h}{x} \cdot x} = \frac{1}{x} \cdot \lim_{\frac{h}{x} \to 0} \frac{\ln(1 + \frac{h}{x})}{\frac{h}{x}} = \frac{1}{x} \cdot 1 = \frac{1}{x}$$Izračun drugog limesa:$$\lim_{h \to 0} (\ln(x+h) + \ln x) = \ln x + \ln x = 2\ln x$$Konačni rezultat:$$f'(x) = \frac{1}{x} \cdot 2\ln x = \frac{2\ln x}{x}$$Zadatak (b)Odredite $f'(0)$ za funkciju $f(x) = x \operatorname{tg} x + (x^2 + 1)^x$.Funkciju možemo promatrati kao zbroj dva dijela: $f(x) = u(x) + v(x)$.Derivacija prvog dijela $u(x) = x \operatorname{tg} x$:Koristimo pravilo za derivaciju produkta:$$u'(x) = (x)' \operatorname{tg} x + x (\operatorname{tg} x)' = 1 \cdot \operatorname{tg} x + x \cdot \frac{1}{\cos^2 x}$$U točki $x = 0$:$$u'(0) = \operatorname{tg} 0 + 0 \cdot \frac{1}{\cos^2 0} = 0 + 0 = 0$$Derivacija drugog dijela $v(x) = (x^2 + 1)^x$:Ovo je funkcija oblika $g(x)^{h(x)}$, pa koristimo identitet $a^b = e^{b \ln a}$:$$v(x) = e^{x \ln(x^2 + 1)}$$Deriviramo koristeći lančano pravilo:$$v'(x) = e^{x \ln(x^2 + 1)} \cdot \left( x \ln(x^2 + 1) \right)'$$$$v'(x) = (x^2 + 1)^x \cdot \left[ 1 \cdot \ln(x^2 + 1) + x \cdot \frac{1}{x^2 + 1} \cdot 2x \right]$$U točki $x = 0$:$$v'(0) = (0^2 + 1)^0 \cdot \left[ \ln(0^2 + 1) + \frac{2 \cdot 0^2}{0^2 + 1} \right]$$$$v'(0) = 1 \cdot [ \ln 1 + 0 ] = 1 \cdot [0 + 0] = 0$$Konačni izračun:$$f'(0) = u'(0) + v'(0) = 0 + 0 = 0$$
